# 成效衡量

所有指標於每次分析後自動計算並寫入 `result.json`（`metrics` 欄位），定義見 `metrics.py`。

## 指標定義
| 指標 | 定義 / 公式 | 意義 |
|---|---|---|
| **節省剪輯時間比例** | `1 − 處理時間 / 人工基準` | 自動化相對人工的省時 |
| 人工基準（秒） | `回看(=時長) + 高光數×720 + 額外平台變體×180` | 可稽核的人工工時模型 |
| **即時處理倍率** | `來源時長 / 處理時間` | 單機吞吐（示範約 3–4×） |
| **高光偵測信心** | 峰值相對中位數的顯著度平均 | 偵測穩健度（非任意挑點） |
| **產出短片品質評分** | `融合分×55 + 多模態廣度×6.5 + 時長適配×16`（0–100） | 預測短片品質 |
| **自動化程度** | 通過審核免人工的短片比例 | 端到端無人介入程度 |
| 內容再利用壓縮比 | `來源時長 / 精華總時長` | 一場直播濃縮成幾倍精華 |
| 審核標記率 | 被標記/需複審的短片比例 | 合規風險可視化 |

## 示範素材實測（70 秒合成直播）
| 指標 | 數值 |
|---|---|
| 高光偵測 | **3/3**（完全對應三個爆點窗口） |
| 節省剪輯時間 | **≈ 99.4%**（人工基準 ≈ 55 分 → 處理 ≈ 18 秒） |
| 即時處理倍率 | **3–4×** |
| 平均品質評分 | **73/100** |
| 自動化程度 | **100%** |
| 內容再利用壓縮比 | **2.5×** |
| 審核結果 | pass（無違規） |

> 註：人工基準採保守工時模型（每支高光約 12 分鐘剪輯 + 每額外平台 3 分鐘匯出 + 回看全片），
> 各工作室可於 `metrics.py` 依實際工時調整常數。

## 商業效益
### 對創作者（變現）
- **產能倍增**：一場直播即時衍生數支短片，跨平台觸及新受眾、導流回主頻道；
- **留存優化**：自動開場鉤子與字幕提升前 3 秒留存與完播率，直接影響演算法推播與廣告分潤；
- **零門檻**：非剪輯專業者也能一鍵產出平台原生內容。

### 對平台 / MCN（內容再利用）
- **長尾活化**：把沉睡的長時數 VOD 批量轉為可再分發的短影音資產（`sparkreel batch`）；
- **合規保障**：三模態審核 + 人工複審閘，降低下架與品牌風險；
- **資料資產**：高光與成效指標入庫（DynamoDB），可回饋推薦與內容策略。

### 量化想像（以每日 100 場直播的中型平台估算）
| 項目 | 人工 | SparkReel |
|---|---|---|
| 單場處理工時 | ~55 分 | ~18 秒（＋人工僅複審少數） |
| 每日可處理場次 | 受限於剪輯師人數 | **近乎無上限（雲端水平擴展）** |
| 每日產出短片 | 數十支 | **數百～上千支** |
